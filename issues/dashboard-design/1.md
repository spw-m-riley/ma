# Reframe the homepage around recent activity with a quiet savings band

## Type

AFK

## User stories covered

1, 2, 3, 7, 12, 13, 14

## What to build

Refine the dashboard landing page so it reads as a balanced live companion instead of a dense status surface. Make recent activity the primary focus, keep aggregate bytes/words/token savings visible in a smaller supporting band, and express the `ma` concept through spacing, hierarchy, and restraint rather than heavy card chrome or control-room styling.

## Acceptance criteria

- [ ] The homepage gives recent runs and current activity stronger visual priority than aggregate value metrics.
- [ ] Bytes, words, and token savings remain visible in a compact supporting band without competing with live activity.
- [ ] The layout uses spacing, typography, subtle dividers, and restrained color to guide attention instead of dense cards or heavy chart framing.
- [ ] The homepage remains easy to scan on narrow and wide screens without feeling visually cluttered.

## Blocked by

None - can start immediately.
