# Give the stats view deeper analysis without losing the calm visual style

## Type

AFK

## User stories covered

3, 8, 9, 12, 14, 15

## What to build

Refine the dashboard's stats view so deeper historical analysis lives there instead of crowding the homepage. Keep the page focused on long-term value through bytes, words, and token savings, command rankings, and outcome context, using restrained micro-indicators and lightweight comparison patterns instead of large chart-heavy treatments.

## Acceptance criteria

- [ ] The stats view carries deeper trends, rankings, and outcome context so the homepage can stay focused on recent activity.
- [ ] Small trend indicators or microchart accents add context without taking over the page with large standalone charts.
- [ ] The visual treatment stays text-first, restrained, and consistent with the homepage and detail views.
- [ ] Any specialized display typography is limited to narrow accent use and does not replace the primary legible reading typography.

## Blocked by

None - can start immediately.
