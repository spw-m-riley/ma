# Specific-purpose extension tools for direct strategy selection

Type: AFK

## What to build

Add specific-purpose tools to the extension alongside the smart-read default, so that the agent or user can select a particular reduction strategy when automatic classification is insufficient. Each tool delegates to the corresponding CLI command and returns results in the standard `app.Result` contract.

Tools to add:

- **ma-compress** — prose compression via the `compress` command
- **ma-skeleton** — code declaration extraction via the `skeleton` command
- **ma-minify-schema** — JSON/YAML schema minification via the `minify-schema` command
- **ma-dedup** — duplicate detection via the `dedup` command

Each tool should respect sensitive-path protections and produce the same results as invoking the corresponding CLI command directly.

## Acceptance criteria

- [ ] Each specific-purpose tool is exposed in the extension and callable by the agent
- [ ] Each tool produces the same results as the corresponding CLI command
- [ ] Each tool respects sensitive-path protections
- [ ] Each tool returns results in the standard `app.Result` contract
