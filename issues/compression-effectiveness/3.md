# Wordy phrases, transitions, and filler adverbs

## Type

AFK

## User stories covered

1, 13

## What to build

Add three rule categories to the phased compression engine: Phase 4 (wordy phrase compression, ~22 rules), Phase 5 (transition simplification, ~8 rules), and Phase 6 (filler adverb removal, ~6 rules). Together, these are the bulk contributor to the 15–30% reduction target on verbose content.

Wordy phrase compression replaces multi-word constructions with concise equivalents: `"in order to"` → `"to"`, `"due to the fact that"` → `"because"`, `"in the event that"` → `"if"`, `"for the purpose of"` → `"for"`, `"whether or not"` → `"whether"`, etc.

Transition simplification reduces or removes connective filler: `"However,"` → `"but "`, `"Furthermore,"` → (empty), `"Therefore,"` → `"so "`, `"Additionally,"` → (empty).

Filler adverb removal strips adverbs that add no information: `"basically"`, `"essentially"`, `"actually"`, `"currently"`, `"specifically"`, `"simply"`.

Each category should have per-category unit tests.

## Acceptance criteria

- [ ] Wordy phrase compression rules (~22) are implemented as Phase 4 and correctly replace verbose constructions with concise equivalents.
- [ ] Transition simplification rules (~8) are implemented as Phase 5 and correctly simplify or remove connective filler.
- [ ] Filler adverb removal rules (~6) are implemented as Phase 6 and correctly strip zero-information adverbs.
- [ ] Per-category unit tests exercise representative patterns for all three categories.
- [ ] Running `ma compress` on a verbose document containing these patterns produces measurably higher reduction than contractions alone.
- [ ] Zone protection and structural preservation remain intact.
